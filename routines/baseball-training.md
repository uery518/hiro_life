# 草野球投手トレーニングルーティン

## 週次スケジュール

週によって2パターンを使い分ける（その週に試合があるかどうかで選ぶ）。

### パターン1：試合がない週

| 曜日 | 内容 |
| --- | --- |
| 月 | 休み |
| 火 | 自宅（上半身＋下半身） |
| 水 | 休み |
| 木 | ジム（下半身） |
| 金 | 休み |
| 土 | ジム（上半身） |
| 日 | ジム（下半身） |

### パターン2：試合がある週

| 曜日 | 内容 |
| --- | --- |
| 月 | 休み |
| 火 | ジム（下半身） |
| 水 | 休み |
| 木 | ジム（上半身） |
| 金 | ジム（下半身） |
| 土 | 休み |
| 日 | 試合 |

**振り分けの考え方:** ジムでしか出せない負荷（高重量バーベル・懸垂バー・ジャンプ台・ケーブルマシン）はジムに集約し、それ以外（ダンベル・バンドで十分な種目）は自宅で行う。

**ジム（下半身）が週2回ある場合:** A（最大筋力＋パワー）とB（ボリューム）を交互に実施する。

---

## ジム（下半身）A：最大筋力＋パワー

| 種目 | セット × 回数 | 備考 |
| --- | --- | --- |
| スクワット（高重量） | 3〜5 × 3〜5 | フル（パラレル以深）。**股関節主導**を意識し、重量より深さの質を優先 |
| RDL（高重量） | 3 × 6〜8 | 臀部・ハムを鍛え球速の土台をつくる |
| ジャンプスクワット／ボックスジャンプ | 3 × 5 | 筋力を爆発力に変換 |
| ケーブルウッドチョッパー | 3 × 8（各方向） | 下半身の力を体幹回旋へ伝達（メディシンボール代替） |
| チンニング | 4〜5 × 限界 | 減速筋（広背筋・肩甲帯）は投球障害予防に重要。ジムの日は毎回実施 |

**スクワット目標推移:**

- 現在: 1RM 100kg（体重比1.37倍）
- Phase 1目標: 110〜115kg（体重比1.5倍）
- Phase 2目標: 120〜125kg（体重比1.7倍）

## ジム（下半身）B：ボリューム

| 種目 | セット × 回数 | 備考 |
| --- | --- | --- |
| スクワット（70〜75%1RM） | 3 × 8 | フル（パラレル以深）。筋肥大・土台強化 |
| ケーブルウッドチョッパー | 3 × 8（各方向） | 回旋パワーの継続刺激・左右バランスを均等に |
| アブローラー | 1〜3 × 限界 | 体幹の安定 |
| チンニング | 4〜5 × 限界 | 自宅バンドでは負荷不足。ジムの日は毎回実施 |

## 毎日のルーティン

| 種目 | 回数・時間 | 備考 |
| --- | --- | --- |
| 肩回し | 前後各50回 | 肩関節の可動域維持・ウォームアップ |

## 毎日のストレッチ（5〜10分・起床後または就寝前）

| 部位 | 種目 | 時間・回数 |
| --- | --- | --- |
| ふくらはぎ | 壁押しカーフストレッチ | 30秒 × 2（両脚） |
| 股関節屈筋 | ランジポジションで前傾 | 30秒 × 2（両脚） |
| 股関節内旋 | 仰向けで膝を内側に倒す | 30秒 × 2（両側） |
| 胸郭回旋 | 四つ這いで肘を天井へ回す | 10回 × 2（両側） |
| 肩 | クロスボディで腕を引き寄せる | 30秒 × 2（両腕） |

## Phase 2 以降に追加予定（2027年〜）

- シングルレッグデッドリフト（片脚の安定性・バランス）

---

## 試合後のケア

- 投球後は必ず肩・肘・前腕のストレッチを10分行う
- アイシングは痛みがある場合のみ（予防的アイシングは不要）

---

## 注意事項

- **肩・肘に違和感があるときは即日投球中止**
- ふくらはぎがつったら無理せず休む
- スクワットの重量より**フォームの質**（フル深さ）を優先する
