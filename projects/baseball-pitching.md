# 草野球：投手球速アップ目標

## サマリー
- **ゴール:** 球速150km/hを目指し、まず段階的に143km/h台へ
- **現状:** 全力投球130km/h、試合では125km/h前後
- **試合頻度:** 月1〜2回
- **現在年齢:** 43歳

---

## 現状の課題
- 球速ポテンシャル: 130km/h（コントロール捨てた場合）
- ふくらはぎがつりやすい → 股関節・臀部が弱く末端に負担が集中している可能性
- 有酸素・ストレッチ・投球特化トレーニングはゼロ
- 筋力トレーニングは実施中（上半身は土台あり）

## 下半身の現状
- 体重: 72〜73kg
- スクワット1RM: 100kg（体重比 **1.37倍**）
- 投球に必要な目安: **1.5〜1.8倍**（110〜130kg）
- → 下半身がボトルネック。ここの強化が球速アップの最優先

## 球速アップに必要な要素
| 要素 | 重要度 | 現状 |
|---|---|---|
| 下半身の爆発力（股関節・臀部） | ★★★ | 不足（スクワット1.37倍） |
| 体幹の回転力（股関節〜肩の分離） | ★★★ | 未強化 |
| 柔軟性・モビリティ | ★★★ | 不足（ふくらはぎに症状あり） |
| 肩周り（ローテーターカフ）のケア | ★★★ | 未実施（43歳、故障予防に必須） |
| 有酸素・スタミナ | ★★ | 未実施 |
| 投球メカニクス | ★★ | 未分析 |

---

## ロードマップ

### Phase 1: 土台を整える（2026-06 〜 2026-12）
下半身強化と故障予防を最優先にしながら、投球特化の要素を追加する。

- [ ] スクワットを週2回実施し、フォームを股関節主導に改善する
- [ ] デッドリフト・RDL（ルーマニアンデッドリフト）を追加する（臀部・ハム強化）
- [ ] ストレッチ・モビリティルーティンを毎日5〜10分行う（ふくらはぎ・股関節・胸郭）
- [ ] ローテーターカフ強化エクササイズを週2回取り入れる（肩の故障予防）
- [ ] 体幹の回転力トレーニングを追加する（メディシンボール回旋・プランク系）
- [ ] 試合後に必ずクールダウンストレッチをする習慣をつける
- **スクワット中間目標:** 100kg → 110〜115kg（体重比1.5倍）
- **球速到達目標:** 130 → 133〜135km/h

### Phase 2: 爆発力・回転力を上げる（2027-01 〜 2027-06）
投球動作に直結する「使い方」を強化する。

- [ ] プライオメトリクス（ジャンプ系）をトレーニングに追加する
- [ ] ヒップヒンジ・回旋動作を意識した下半身トレーニングに切り替える
- [ ] 可能であれば1回だけ投球フォームを動画撮影して確認する
- **到達目標:** 135 → 138〜140km/h

### Phase 3: 仕上げと定着（2027-07 〜 2028-06）
- [ ] 試合での球速を意識的に計測・記録する
- [ ] フォームの細かい修正（必要に応じてコーチングを受ける）
- [ ] 140km/h超を試合でも安定して出せるようにする
- **到達目標:** 140 → 143km/h台、安定化

---

## 注意事項
- **43歳のため肩・肘のケアを最優先にする** — 故障したら終わり
- 球速より「健康に長く投げ続けること」が前提
- ふくらはぎのつりは放置しない（柔軟性・水分補給・下半身強化で対処）
